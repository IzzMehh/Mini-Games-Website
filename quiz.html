Gameplay

  1. About Game:
    • You have to give answer of 20 questions (5 easy/ 5 medium/ 10 hard)

    • You will get some amout of money for each question and bonus according to the deficulty level

  2. Help Options:
    • The game provides 50-50 Option in which a user can remove 2 wrong options from the game.

    • The game provides call a friend Option in which a random call box will appear and the a random person will tell you the answer.

    • The game provides a Majority Poll option in which a graph will apear with random winning percentage of each option.

    • The game provides a skip option in which the game will automatically select the right option for you.

    • Remember each option can only be used once in a game. <3
